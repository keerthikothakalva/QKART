# QKart – Full Stack E-Commerce Platform

## Overview

QKart is a full-stack e-commerce platform built using the MERN stack that enables users to browse products, manage carts, and complete secure checkout workflows. The application focuses on scalability, security, and seamless user experience.

## Features

* User authentication and authorization
* Product browsing and search
* Shopping cart management
* Secure checkout workflow
* Responsive user interface
* RESTful API architecture
* Password hashing and JWT authentication
* Cloud-hosted deployment
